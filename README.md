# C++-Bookmark_Manager
網頁書籤管理程式

撰寫一個網頁書籤管理程式
– 使用 OOP 的方式實作 , 使用 C++ Class, inheritance, polymorphism, template 等特性
– 網頁書籤的屬性包含
  • 書籤名稱 (Name)
  • 書籤網址 (URL)
  • 標籤 (Tags): 可以有多個標籤 , 用 ";" 分隔
  • 註解 (Comment)
– 一個資料夾可以為空 , 或者包含多個書籤或資料夾
– 可以自動將書籤管理資料儲存到檔案 , 程式啟動後自動載入

提供文字介面以便進行操作
– 能夠新增、修改、刪除或開啟書籤
– 能夠新增、刪除或開啟資料夾
– 能夠以標籤 (Tag) 搜尋書籤，並條列出來讓使用者選擇
– 以樹狀的選單形式來瀏覽整個書籤

可直接開啟指定網址
